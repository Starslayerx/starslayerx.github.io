### 亚导数

- 将导数扩展到不可微的函数

例如函数 $y=\|x\|$ 亚导数可以这样定义

$$
\frac{\partial |x|}{\partial x} =
\begin{cases}
1, & \text{if } x > 0 \\\\
-1, & \text{if } x < 0 \\\\
a, & \text{if } x = 0, a \in [-1,1]
\end{cases}
$$

另一个例子

$$
\frac{\partial }{\partial x}max(x, 0)=
\begin{cases}
1, & \text{if } x > 0 \\\\
0, & \text{if } x < 0 \\\\
a, & \text{if }, a \in [0,1]
\end{cases}
$$
